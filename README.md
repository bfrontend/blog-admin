# blog-admin
基于iview的后台管理系统


#### 项目介绍
个人博客的后端管理界面,采用前后分离

#### 软件架构
iview vue simplemde wangeditor echarts axios mock sendemail jwt OAuth2.0 mosquitto


#### 安装教程

1. git clone https://github.com/webxmsj/blog-admin.git xxx
2. cd xxx && cnpm i
3. npm run dev

#### 使用说明

1. scp ./xxx root@baidu.com:/var/www/
2. pm2 start pm2.json
